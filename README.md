# MealApp

# Functionality:
1-App get data from the hardcoded Lists of Meal Items.<br>
2-Categories the Meals.<br>
3-Filters the Meals as Gluton Free,Lactose Free, Vegetarian and other filters.<br>
4-Mark the Meals as Favorite.<br>
